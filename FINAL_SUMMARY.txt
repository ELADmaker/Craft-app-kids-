🚀 DÉPLOIEMENT - 3 OPTIONS
═══════════════════════════════════════════════════════════════════════════

OPTION 1: Drag & Drop (Recommandé pour débutants)
─────────────────────────────────────────────────
⏱️  Temps : 2 minutes
📊 Difficulté : Très facile
🔗 Lien : https://netlify.com

Étapes :
1. Allez sur netlify.com
2. Cliquez "Add new site" → "Deploy manually"
3. Glissez-déposez ce dossier
4. Votre site est en ligne ! 🎉

URL générée : https://cryptomine-XXXXX.netlify.app


OPTION 2: GitHub (Recommandé pour professionnels)
──────────────────────────────────────────────────
⏱️  Temps : 5 minutes
📊 Difficulté : Facile
✨ Avantages : Auto-déploiement, CI/CD, historique

Étapes :
1. Créez un repo GitHub : https://github.com/new
2. Poussez le code :
   git remote add origin https://github.com/USERNAME/cryptomine-website.git
   git branch -M main
   git push -u origin main
3. Sur Netlify, cliquez "New site from Git"
4. Sélectionnez votre repo
5. Netlify déploiera automatiquement ! 🎉


OPTION 3: Netlify CLI (Pour développeurs)
──────────────────────────────────────────
⏱️  Temps : 3 minutes
📊 Difficulté : Moyen
✨ Avantages : Contrôle total

Commandes :
npm install -g netlify-cli
netlify login
netlify deploy --prod --dir=.
