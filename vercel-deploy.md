# Déploiement Automatisé avec Vercel (Alternative Gratuite)

## 🚀 Vercel - Alternative Gratuite à Netlify

Vercel est une plateforme de déploiement gratuite et très simple, créée par les makers de Next.js.

### Avantages de Vercel :
- ✅ Gratuit et illimité
- ✅ Déploiement en 1 clic
- ✅ HTTPS automatique
- ✅ Domaine gratuit
- ✅ Très rapide
- ✅ Intégration GitHub

---

## 📋 Étapes de Déploiement avec Vercel

### 1. Créer un Compte Vercel

1. Allez sur **https://vercel.com**
2. Cliquez **"Sign Up"**
3. Connectez-vous avec GitHub (recommandé) ou email
4. Confirmez votre email

### 2. Importer le Projet

1. Cliquez **"New Project"**
2. Sélectionnez **"Import Git Repository"**
3. Connectez votre repo GitHub (ou créez-en un)
4. Cliquez **"Import"**

### 3. Configuration

1. **Project Name** : `cryptomine-website`
2. **Framework** : Sélectionnez **"Other"** (site statique)
3. **Root Directory** : `.` (racine)
4. Cliquez **"Deploy"**

### 4. Voilà !

Votre site est maintenant en ligne sur :
```
https://cryptomine-website.vercel.app
```

---

## 🌐 Ajouter un Domaine Personnalisé

1. Dans Vercel → **Settings** → **Domains**
2. Cliquez **"Add Domain"**
3. Entrez votre domaine (ex: `cryptomine.com`)
4. Suivez les instructions DNS
5. Attendez 24-48h

---

## 🔄 Déploiement Automatique

Chaque fois que vous poussez du code sur GitHub :
```bash
git push origin main
```

Vercel déploiera automatiquement ! 🎉

## 🎯 Recommandation

**Utilisez Vercel si :**
- Vous avez un compte GitHub
- Vous voulez un déploiement automatique
- Vous préférez une interface simple

**Utilisez Netlify si :**
- Vous préférez le drag & drop
- Vous voulez plus de contrôle
- Vous avez besoin de Netlify Forms
