### Option 3: GitHub Pages (⭐ RECOMMANDÉ)

**Temps**: 4 minutes | **Difficulté**: Facile | **Coût**: Gratuit

1. Créez un repo GitHub : `cryptomine-website`
2. Poussez le code :
   ```bash
   cd /home/ubuntu/cryptomine-website
   git remote add origin https://github.com/USERNAME/cryptomine-website.git
   git branch -M main
   git push -u origin main
   ```
3. Allez dans **Settings** → **Pages**
4. Sélectionnez **"Deploy from a branch"** → **main**
5. Cliquez **"Save"**
6. **Votre site est en ligne ! 🎉**

**URL générée** : `https://USERNAME.github.io/cryptomine-website`

## 🌐 DOMAINE PERSONNALISÉ (Optionnel)

### Acheter un domaine

1. Allez sur **namecheap.com**, **godaddy.com**, ou **domains.google.com**
2. Cherchez `cryptomine.com` (ou votre domaine préféré)
3. Achetez pour 1 an (~$10-15)
4. Confirmez votre email

### Configurer le domaine

**Sur Netlify** :
1. Settings → Domain Management
2. Cliquez "Add custom domain"
3. Entrez votre domaine
4. Suivez les instructions DNS
5. Attendez 24-48h

**Sur Vercel** :
1. Settings → Domains
2. Cliquez "Add"
3. Entrez votre domaine
4. Configurez les DNS
5. Attendez 24-48h

**Sur GitHub Pages** :
1. Settings → Pages
2. Entrez votre domaine dans "Custom domain"
3. Configurez les DNS chez votre registraire
4. Attendez 24-48h
